# Ambient beds (sleep stories)

Drop ambient soundscape files here to offer them under the **Sleep Stories**
content type. They are mixed *under* the narration — looped/trimmed to the story
length, pulled well below the voice, and faded in/out.

```
assets/ambient/<name>.wav     # or .mp3
```

Rules:

- Any `.wav` or `.mp3` file is offered. The slug is the filename stem; the
  display name is the slug title-cased (`gentle_rain` → "Gentle Rain").
- The bed is looped automatically, so a short seamless loop (30–120 s) works
  well for a 30–45 min story.
- Stereo files are recommended (sleep masters are 44.1 kHz stereo).
- Restart the backend after adding files; the bed appears in the ambient picker.

Ambient mixing applies to sleep stories only — podcasts are never processed.
